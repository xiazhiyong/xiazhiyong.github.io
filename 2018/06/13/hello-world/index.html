如何快速创建一个hexo搭建的博客

欢迎你来到我的博客:
这是我的第一篇文章这篇文章将讲述如何搭建一个由hexo搭建的博客并发布到github给我们的仓库中。.

准备工作

  • 1.申请github账号点击这里进入github

  • 2.创建仓库进行操作https://github.com/new

  • 3.生成密钥git工具生成密钥放入github中

  • 4.密钥生成方法 $ ssh-keygen -t rsa -C “youremail@example.com“

  • 4.安装hexo cnpm install -g hexo-cli

  • 5.初始化 hexo init blog

  • 6. 发布 $ hexo deploy
